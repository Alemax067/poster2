# HTML 海报导出工具

## 使用方法

1. 安装依赖

```bash
pip install -r requirements.txt
```

2. 运行程序

```bash
python poster_exporter.py
```

3. 访问 http://localhost:7860 查看界面

# 环境配置
## Windows + CMD终端
```bash
pip install -r requirements.txt

python poster_exporter.py
```

## Windows + WSL终端
```bash
pip install -r requirements.txt

playwright install-deps chromium

python poster_exporter.py
```

## Linux + Terminal
```bash
pip install -r requirements.txt

playwright install-deps chromium

python poster_exporter.py
```

## macOS
```bash
pip install -r requirements.txt
python poster_exporter.py
```
